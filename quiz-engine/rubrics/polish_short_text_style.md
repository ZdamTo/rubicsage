# Rubric: Matura z Języka Polskiego — Odpowiedź na pytanie do tekstu (CKE Style)

**Scoring: 0 to maxScore points (typically 1–2 per question)**

## Criteria

### Content / key points (0 – maxScore pkt)
| Points | Description |
|--------|------------|
| maxScore | Response addresses all expected key points. Each point is clearly stated and supported by reference to the text. |
| partial | Some key points are addressed; others are missing or vaguely stated. Award proportionally. |
| 0 | No key points addressed, or answer is off-topic / incomprehensible. |

## Rules
- The student must **refer to the provided text** — general knowledge alone is insufficient.
- Accept paraphrases and synonymous expressions.
- Minor language errors do not reduce the score unless they make the answer unintelligible.
- If the question asks for N reasons/examples and the student provides fewer, score proportionally.
