# Rubric: Matura z Języka Polskiego — Wypracowanie (Poziom Podstawowy, 2025)

**Total: 35 points**

## A. Wymagania formalne (0–1 pkt)

| Punkty | Opis |
|--------|------|
| 1 | Praca spełnia wymagania formalne: minimum 300 wyrazów, jest wypowiedzią argumentacyjną (rozprawka), odnosi się do tematu. |
| 0 | Praca nie spełnia wymagań formalnych (np. poniżej 250 wyrazów, nie jest rozprawką, nie odnosi się do tematu). Jeśli 0 punktów tu, cała praca = 0 pkt. |

## B. Kompetencja literacka i kulturowa (0–16 pkt)

### B1. Odwołanie do lektury obowiązkowej (0–8 pkt)
| Punkty | Opis |
|--------|------|
| 8 | Funkcjonalne, pogłębione odwołanie do lektury obowiązkowej. Przywołuje konkretne sytuacje, postacie, cytaty. Analiza i interpretacja służą argumentacji. |
| 6 | Funkcjonalne odwołanie do lektury obowiązkowej, choć mniej pogłębione. Przywołane elementy są trafne. |
| 4 | Odwołanie do lektury obowiązkowej jest częściowo funkcjonalne. Przywołane elementy są ogólnikowe lub niewystarczająco powiązane z tezą. |
| 2 | Odwołanie do lektury obowiązkowej jest powierzchowne lub zawiera istotne błędy rzeczowe. |
| 0 | Brak odwołania do lektury obowiązkowej lub odwołanie całkowicie błędne. **Jeśli 0 pkt tu, cała praca = 0 pkt.** |

### B2. Odwołanie do drugiego utworu + kontekst (0–8 pkt)
| Punkty | Opis |
|--------|------|
| 8 | Funkcjonalne, pogłębione odwołanie do wybranego utworu literackiego/kontekstu kulturowego. Przywołanie wspiera argumentację. Zastosowany kontekst (literacki, historyczny, filozoficzny, etc.) jest trafny. |
| 6 | Odwołanie do drugiego utworu jest funkcjonalne; kontekst obecny i trafny. |
| 4 | Odwołanie i kontekst są obecne, ale niepełne lub słabo powiązane z argumentacją. |
| 2 | Odwołanie lub kontekst są powierzchowne lub częściowo błędne. |
| 0 | Brak odwołania do drugiego utworu lub brak kontekstu. |

## C. Kompozycja, spójność, styl (0–7 pkt)

### C1. Kompozycja i spójność (0–4 pkt)
| Punkty | Opis |
|--------|------|
| 4 | Praca jest logicznie skomponowana: wstęp z tezą, rozwinięcie z argumentami, zakończenie z podsumowaniem. Spójna, przejrzysta struktura. |
| 3 | Kompozycja w większości poprawna, drobne uchybienia spójności. |
| 2 | Kompozycja częściowo zaburzona; widoczne problemy ze spójnością lub logicznym porządkiem. |
| 1 | Kompozycja poważnie zaburzona; tekst chaotyczny. |
| 0 | Brak rozpoznawalnej kompozycji. |

### C2. Styl (0–3 pkt)
| Punkty | Opis |
|--------|------|
| 3 | Styl konsekwentny, stosowny do formy rozprawki. Język precyzyjny i urozmaicony. |
| 2 | Styl w większości stosowny, z drobnymi uchybieniami. |
| 1 | Styl niekonsekwentny lub miejscami niestosowny (potoczny, zbyt emocjonalny). |
| 0 | Styl rażąco niestosowny. |

## D. Język (0–11 pkt)

### D1. Poprawność językowa (0–6 pkt)
| Punkty | Opis |
|--------|------|
| 6 | Praca wolna od błędów językowych (gramatycznych, leksykalnych, frazeologicznych, fleksyjnych, składniowych). |
| 5 | Sporadyczne błędy językowe (1–2). |
| 4 | Nieliczne błędy językowe (3–4). |
| 3 | Błędy językowe (5–6), ale nie utrudniają rozumienia. |
| 2 | Liczne błędy językowe (7–8), mogą utrudniać rozumienie. |
| 1 | Bardzo liczne błędy językowe (9+). |
| 0 | Błędy językowe uniemożliwiają rozumienie tekstu. |

### D2. Poprawność ortograficzna (0–3 pkt)
| Punkty | Opis |
|--------|------|
| 3 | Bezbłędna ortografia. |
| 2 | 1–2 błędy ortograficzne. |
| 1 | 3–4 błędy ortograficzne. |
| 0 | 5+ błędów ortograficznych. |

### D3. Poprawność interpunkcyjna (0–2 pkt)
| Punkty | Opis |
|--------|------|
| 2 | Bezbłędna interpunkcja lub sporadyczne uchybienia (1–2). |
| 1 | Kilka błędów interpunkcyjnych (3–5). |
| 0 | Liczne błędy interpunkcyjne (6+). |

---

## WAŻNA ZASADA

Jeśli za kryterium B1 (odwołanie do lektury obowiązkowej) uczeń otrzyma **0 punktów**, to za całą pracę przyznaje się **0 punktów** — niezależnie od jakości pozostałych elementów.

Analogicznie: jeśli kryterium A (wymagania formalne) = 0, cała praca = 0 pkt.
