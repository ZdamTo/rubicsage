# Rubric: Matura z Matematyki — Zadanie otwarte (CKE Style, 2025)

**Scoring: 0 to maxScore points (typically 2–5 per task)**

## General Criteria

### Method and reasoning (0 – ceil(maxScore * 0.5) pkt)
| Level | Description |
|-------|------------|
| Full | Correct method chosen, reasoning is complete and logically presented. |
| Partial | Correct method chosen, but reasoning has minor gaps or is incomplete. |
| Minimal | An attempt at a correct method is visible, but execution is largely flawed. |
| 0 | No correct method shown, or work is completely off-topic. |

### Computation / execution (0 – floor(maxScore * 0.5) pkt)
| Level | Description |
|-------|------------|
| Full | All computations are correct, final answer matches expected result. |
| Partial | Minor arithmetic/algebraic error, but the method would lead to a correct result. |
| 0 | Major errors or no computation shown. |

## Special rules
- If the final answer is correct but no work is shown, award at most 1 point (for tasks worth >= 2 pts).
- If the method is entirely wrong but the final answer is somehow correct (lucky guess), award 0.
- Accept equivalent forms of the answer (e.g., fractions vs decimals) unless the problem specifies a form.
